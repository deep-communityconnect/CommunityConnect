# CommunityConnect

Welcome to **CommunityConnect**, a platform designed to streamline and improve volunteer management, community organizing, and user engagement!

## 🚀 Overview

CommunityConnect is a full-stack web application built to facilitate connections within local communities. It features dedicated modules for organizers to manage events, a robust system for users to volunteer, and authentication capabilities.

## 💻 Tech Stack

- **Frontend**: React.js, Node.js
- **Backend**: Python, Django
- **Testing Tools**: Postman

## 📁 Project Structure

The project is structured as a monorepo containing both the frontend and the backend parts of the application:

- **`/frontend`**: Contains the React application (UI, components, styles, etc.).
- **`/backend`**: Contains the Django application (API, models, serializers, etc.) including components for Users, Organizers, and the Volunteer Platform.
